## Date

## Goal

Start the NucleOS educational operating system project.

## Project Purpose

NucleOS is an educational operating system project created to learn low-level computing, operating systems, boot processes, memory management, interrupts, and kernel development.

The objective is not to replace Linux or build a production-ready operating system.

The objective is to learn how computers work from the boot process to the kernel.
