# NucleOS 60-Day Roadmap

## Phase 0 — Foundation

Days 0–10: project structure, tools, documentation, QEMU, OSDev basics.

## Phase 1 — First Kernel

Days 11–20: first bootable kernel.

## Phase 2 — Organized Kernel

Days 21–30: cleaner kernel structure and basic output.

## Phase 3 — Interrupts and Keyboard

Days 31–40: IDT, exceptions, timer, keyboard.

## Phase 4 — Memory

Days 41–50: memory concepts, paging, heap basics.

## Phase 5 — Minimal Shell

Days 51–60: simple commands, final documentation, demo.
