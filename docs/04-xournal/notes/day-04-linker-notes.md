## Title

Linker Notes

## Notes to Write by Hand

Compiler:

Translates C code into object files.

Linker:

Combines object files into kernel.elf.

Linker script:

Controls the memory layout.

Entry point:

The first place where the kernel begins execution.

Important idea:

The kernel cannot be built like a normal app because it runs without an operating system.
